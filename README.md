# 2D Graphics Editor

## Description
This mini project is a simple 2D Graphics Editor developed in C using a 2D character array as a canvas. The program allows users to draw basic shapes using '*' characters and display them on the screen.

## Features
- Draw Rectangle
- Draw Line
- Draw Triangle
- Draw Circle
- Display Canvas
- Clear Canvas
- Delete Selected Area
- Modify Existing Drawing Area

## Technologies Used
- C Programming Language
- 2D Character Array

## Project Objective
To understand the use of arrays, functions, loops, and basic graphics representation using characters in C programming.
